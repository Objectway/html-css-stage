Attribute selectors



p[class="pippo"] { /* Exact attribute */ }

p[class~="pippo"] { /* One word attribute */ }

p[class|="pippo"] { /* One word "-" insensitive */ }

p[class^="pippo"] { /* Prefixed */ }

p[class$="pippo"] { /* Subfixed */ }

p[class*="pippo"] { /* Contained */ }

p[class="pippo" i] { /* Case insensitive */ }

jQuery

What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library. It makes things like DOM, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

What is the DOM?

Well, the DOM stands for "Document Object Model"

Basically is the API through which You can parse and manipulate the Document via javascript.

What does jQuery?

  • Focus on the interaction between JavaScript and HTML
  • (Almost) every operation boils down to:
  • Find some stuff
  • Do something to it

Only one function

Absolutely everything (almost) starts with a call to the jQuery() function. Since it’s called so often, the $ variable is set up as an alias to jQuery


               jQuery('#nav');
               jQuery('div#intro h2');
               jQuery('#nav li.current a');
a.k.a. 

               $('#nav');
               $('div#intro h2');
               $('#nav li.current a');

jQuery Collections

$('div.section') returns a jQuery collection
You can call treat it like an array 


  $('div.section').length // no. of matched elements
  $('div.section')[0] //the first div DOM element

jQuery Collections

$('div.section') returns a jQuery collection
You can call methods on it 


  $('div.section').size()
  $('div.section').each(function(i) {
    console.log("Item " + i + " is ", this);
  });

jQuery Chaining

Most jQuery methods return another jQuery object - usually one representing the same collection.This means you can chain methods together: 


  $('div.section').hide().addClass('gone');

Traversing the DOM


$('div.section').parent();

$('div.section').prev();

$('h1:first').parents();
